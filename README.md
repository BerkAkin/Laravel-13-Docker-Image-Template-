# Laravel Docker Development Template

*A Docker-based development environment designed for modern Laravel applications.*

## Purpose

**This template aims to provide a fast and repeatable development environment by bringing together the essential services needed in the Laravel development process under a single Docker Compose structure.**

## Tech Stack

### PHP 8.4

Laravel is the PHP environment in which the application runs

### Laravel 13

It forms the backend layer of the application.

### Nginx

It receives HTTP requests and redirects them to PHP-FPM.


### MySQL 8

It is the application's relational database service.

### Redis 8

It is used for caching, queuing, and session operations.

### Node.js & NPM

It is used for managing frontend dependencies and running development tools.

### Vite

It provides frontend asset compilation and Hot Module Replacement (HMR) support.

### Bootstrap 5

It is a CSS framework used in user interface development.

### Docker & Docker Compose

It enables all services to run and be managed within isolated containers.
